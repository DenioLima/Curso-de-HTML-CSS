Lista ordenadas

Quero fazer um lista ordenada começando pelo numero 1, para isso vou usar a tag <ol> e dentro de OL <li> para enumerar.

  1. um
  2. dois
  3. tres
  4. quatro
  5. cinco
  6. seis
  7. sete
  8. oito
  9. nove
  10. dez

E posso mudar esse numeros por letras ou numeros romanos por exemplo usando type depois de OL

lista não ordenadas

Para fazer uma sem ter que ter algum tipo de ordem, cou usar a tag <ul>, e depois da mesma forma das ordenas, colocar li dentro lo, e posso usar type da mesma forma.

  • um
  • dois
  • tres
  • quatro
  • cinco
  • seis
  • sete
  • oito
  • nove
  • dez
