Modelos de caixas

Exemplo de caixa box-level

Quero fazer uma caixa e para isso vou usar o css, vou fazer no modelo box-level, onde essa caixa muda de linha automaticamente. Vou usar o width para largura e height para altura,border-width para a largura da borda, border-style:" " para o stilo de linha da borda e border-color:" " para a cor da linha. Vou usar o padding: para dar distancia do conteudo interno com a borda. A margin: é dar distancia do lado de fora da borda, e para deixar a caixa centralizada na tela se usa margin: auto;.O outline: funciona da mesma forma do border. Os codigos no css dessse titulo não estão de forma simplicadas.Posso fazer esse box box-level virar um inline-level colocando display:inline-level;

Exemplo de caixa inline-level

Quero fazer uma caixa e para isso vou usar o css,esse modelo a caixa continua na mesma linha, SGA. Todos os codigos dessa tag âncora estão em forma simplicados no css.Posso fazer esse inline-level virar um box box-level colocando display:block;
