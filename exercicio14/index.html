Elementos de tabela

Quero fazer uma tabela com país,estado,cidade e nome. Para isso vou usar a tag <table>, dentro dessa tag vou usar esse outro tag <caption> para o titulo, vou usar a tag <thead> para o cabeçalho, <tr> e <th> para os itens do cabeçalho da tabela. O corpo da tabela vai ficar dentro da tag <tbody>, e segue tr porem agora a tag <td>
